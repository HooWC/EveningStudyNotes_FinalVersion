2. ### 自动解析错误日期语法

```
=IF(
   DATE(VALUE(RIGHT(H19,4)),
        VALUE(MID(H19,FIND("/",H19)+1,FIND("/",H19,FIND("/",H19)+1)-FIND("/",H19)-1)),
        VALUE(LEFT(H19,FIND("/",H19)-1))
   )
   <=
   DATE(VALUE(RIGHT(N19,4)),
        VALUE(MID(N19,FIND("/",N19)+1,FIND("/",N19,FIND("/",N19)+1)-FIND("/",N19)-1)),
        VALUE(LEFT(N19,FIND("/",N19)-1))
   ),
   "Date Correct",
   "Date Incorrect"
)
```

3. ### 删除重复项

```
方法1：删除重复项（简单做法）

1. 假设你的数据在 Sheet1 的 A 列。
2. 复制这一列到 Sheet2。
3. 在 Sheet2，选中那一列 → 数据(Data) → 删除重复项(Remove Duplicates)。
```

4. 总加

```
📌 数学与统计

=SUM(A1:A10) → 求和

=AVERAGE(A1:A10) → 平均值

=MAX(A1:A10) → 最大值

=MIN(A1:A10) → 最小值

=ROUND(A1,2) → 四舍五入到小数点后 2 位

=COUNT(A1:A10) → 统计数字个数

=COUNTA(A1:A10) → 统计非空单元格数量
```

```
📌 文本处理

=CONCAT(A1,B1) → 合并文本

=LEFT(A1,3) → 取左边 3 个字符

=RIGHT(A1,4) → 取右边 4 个字符

=MID(A1,2,5) → 从第 2 位开始取 5 个字符

=LEN(A1) → 字符串长度

=TRIM(A1) → 删除多余空格

=UPPER(A1) → 转大写

=LOWER(A1) → 转小写
```

```
📌 查找与引用

=VLOOKUP(1001,A2:D100,3,FALSE) → 查找 1001 并返回第 3 列的值

=HLOOKUP(1001,A1:Z10,4,FALSE) → 横向查找

=INDEX(A2:C10,2,3) → 返回第 2 行第 3 列的值

=MATCH(200,B2:B100,0) → 返回 200 在范围中的位置
```

```
📌 日期与时间

=TODAY() → 当前日期

=NOW() → 当前日期和时间

=YEAR(A1) → 提取年份

=MONTH(A1) → 提取月份

=DAY(A1) → 提取日

=WEEKDAY(A1,2) → 星期几（1=周一，7=周日）

=DATEDIF(A1,B1,"d") → 相差天数
```
