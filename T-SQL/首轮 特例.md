```
打开 SQL Server Configuration Manager
```

```
SQL Server Services -> SQL Server (MSSQLSERVER)  -- 确保有启动
```

```
SQL Server Network Configuration > Protocols for MSSQLSERVER

确认 TCP/IP 已启用（Enable）。

双击 TCP/IP → IP Addresses → 确认 IPAll 里的 TCP Port = 1433。

重启 SQL Server 服务。 (SQL Server Services -> SQL Server (MSSQLSERVER) Restart)
```
